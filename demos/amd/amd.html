Note: The "Read Only" toggle will only affect elements with text <input /> elements. If there is an underlying <select /> element, this will remain editable, since these elements do not support the readonly property in HTML5. Thus only the combo-selects, auto-completes and date-select will be affected, and only the date-select will have it's drop-down disabled.
